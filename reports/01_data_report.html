n_offcial_charts <- sum(poptrag$source.officialcharts, na.rm = TRUE)

We ground our analysis on a dataset with 254,334 tracks that were popular in Germany within the last 50 years (PopTraG). The tracks were collected from the following sources:

  • 215,702 tracks from the weekly MediaControl single- and album charts from 18.11.1977 to 21.11.2024 (scraped and connected to corresponding Spotify ids with a linking success rate of 75% for both distinct tracks in singles charts and distinct albums in album charts).

  • 26,949 tracks from Spotify recommendations for 126 genre seeds for the German market (retrieved on 29.11.2024, 5 iterations with 50 recommendations per seed, median of 325 recommendations per seed, IQR: 25, max: 343 for “disney”, min: 49 for “new release”).

  • 8,801 tracks from 100 Spotify featured playlists with median 90 tracks each (IQR: 42, min: 28, max: 750) for the German market (retrieved on 29.11.2024).

  • 11,994 tracks from the Daily Top 200 Spotify charts for Germany from 01.01.2017 to 30.11.2024.

Note

The dataset excluded songs shorter than 30 seconds (most likely interludes; 853 tracks in the sources). Additionally, only distinct combinations of track titles and principal artist names were permitted, while earlier release dates were preferred.

  dplyr::ungroup() |>
  dplyr::pull(n_votes)
tracks_per_tag_mb <- dplyr::count(mb_long, tag_name) |> dplyr::pull(n)

We filtered out all songs without a genre tag and songs with only non-music genre or tags that were not on the Musicbrainz genre whitelist on June 16, 2025 (42,982 tracks, 17%). Non-music genre tags were: “non-music”, “interview”, “comedy”, “nature sounds”, “children’s music”, “spoken word”, “standup comedy”.

This amounts to NMusicBrainz = 211,352 tracks with MusicBrainz music genre tags.

There are 996 different genre tags in the MusicBrainz genre data set. Users assigned between 1 and 25 different tags to each track (median: 3, IQR: 4) and voted for each track between 1 and 156 times (median: 5, IQR: 7).

108 tags were assigned to only one track (11%, min: 1, max: 75,678, median: 54, IQR: 316).

We filtered out all songs without a Discogs genre tag (Discogs “genre” and/or “style”) or with only non-music genre tags (27,269 tracks, 11%). Non-music genre tags were: “Non-Music”, “Children’s”, “Audiobook”, “Comedy”, “Speech”, “Spoken Word”.

This amounts to NDiscogs = 227,065 tracks with Discogs music genre tags.

There are 491 different genre tags in the Discogs genre data set. Users assigned between 1 and 30 different tags to each track (median: 3, IQR: 2).

28 tags were assigned to only one track (6%, min: 1, max: 118,212, median: 102, IQR: 599.5).

Important

Discogs genre tags are assigned to releases, not to tracks. Therefore, all tracks on the same release have the same Discogs genre tags.

Note

In addition to releases being the observational unit, Discogs provides no vote counts for tags on track basis. Thus, we calculated the vote for each track’s tags by counting the number of appearences of a tag within the corresponding artist.

We filtered out all songs without a Spotify genre tag (Spotify artist genre tags) or with only non-music genre tags (1,147 tracks, 0%). Non-music genre tags were: “432hz”, “528hz”, “asmr”, “binaural”, “children’s folk”, “children’s music”, “clean comedy”, “comic”, “escape room”, “field recording ambient”, “football”, “lo-fi sleep”, “lo-fi study”, “mediation”, “sleep”, “talent show”, “talentschau”, “kindermusik”.

This amounts to NSpotifys = 253,187 tracks with Spotify music genre tags.

There are 10,695 different genre tags in the Spotify genre data set. Users assigned between 1 and 19 different tags to each track (median: 1, IQR: 0).

3329 tags were assigned to only one track (31%, min: 1, max: 31,567, median: 3, IQR: 17).

Important

Spotify genre tags are assigned to artists, not to tracks. Therefore, all tracks by the same artist have the same Spotify genre tags.

Note

In addition to artists being the observational unit, Spotify provides no vote counts for tags on track basis. However, some track feature multiple artists. Thus, we calculated the vote for each track’s tags by combining all track artists’ tags and counting the number of tag appearences per track. 33,010 tracks (13%) featured multiple artists.
