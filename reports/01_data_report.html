We ground our analysis on a dataset with 254,334 tracks that were popular in Germany within the last 50 years (POPTRAG, Popular Tracks in Germany). The tracks were collected from the following sources:

  • 215,702 tracks from the weekly MediaControl single- and album charts from 18.11.1977 to 21.11.2024 (scraped and connected to corresponding Spotify ids with a linking success rate of 75% for both distinct tracks in singles charts and distinct albums in album charts).

  • 26,949 tracks from Spotify recommendations for 126 genre seeds for the German market (retrieved on 29.11.2024, 5 iterations with 50 recommendations per seed, median of 325 recommendations per seed, IQR: 25, max: 343 for “disney”, min: 49 for “new release”).

  • 8,801 tracks from 100 Spotify featured playlists with median 90 tracks each (IQR: 42, min: 28, max: 750) for the German market (retrieved on 29.11.2024).

  • 11,994 tracks from the Daily Top 200 Spotify charts for Germany from 01.01.2017 to 30.11.2024.

Note

The dataset excluded songs shorter than 30 seconds (most likely interludes; 853 tracks in the sources). Additionally, only distinct combinations of track titles and principal artist names were permitted, while earlier release dates were preferred.

We filtered out all songs without a genre tag and songs with only non-music genre or tags that were not on the Musicbrainz genre whitelist on June 16, 2025. Additionally, we excluded tags that did not occur with at least 21 artists, as these do not classify for a genre (as no community can be assumed for these tags). Also, we only considered tracks with at least 2 votes to avoid noise from single opinions. This way we removed 85,083 tracks from the data set (33%). Non-music genre tags were: “non-music”, “interview”, “comedy”, “nature sounds”, “children’s music”, “spoken word”, “standup comedy”.

This amounts to NMusicBrainz = 169,251 tracks with MusicBrainz music genre tags.

There are 357 different genre tags in the MusicBrainz genre data set.

If we could not find tags assinged directly to the individual track, we used the tags assinged to the corresponding release. If there were no release tags, we used the artist tags. For 90,871 tracks (54%) tracks the tagging originated from track taggings, for 6,028 tracks (4%) from release taggings, and for 72,352 tracks (43%) from artist taggings.

Users assigned between 1 and 23 different tags to each track (median: 4, IQR: 4) and voted for each track between 2 and 151 times (median: 6, IQR: 7).

We filtered out all songs without a Spotify genre tag (Spotify artist genre tags) or with only non-music genre tags. Additionally, we excluded tags that did not occur with at least 21 artists, as these do not classify for a genre (as no community can be assumed for these tags). This way we removed 33,132 tracks from the data set (13%).Non-music genre tags were: “432hz”, “528hz”, “asmr”, “binaural”, “children’s folk”, “children’s music”, “clean comedy”, “comic”, “escape room”, “field recording ambient”, “football”, “kabarett”, “lo-fi sleep”, “lo-fi study”, “mediation”, “sleep”, “talent show”, “talentschau”, “kindermusik”, “rain”, “white noise”.

This amounts to NSpotify = 221,202 tracks with Spotify music genre tags.

There are 749 different genre tags in the Spotify genre data set. Users assigned between 1 and 22 different tags to each track (median: 3, IQR: 2).

Track-based statistics on tag assignement:

Artist-based statistics on tag assignement:

Important

Spotify genre tags are assigned to artists, not to tracks. Therefore, all tracks by the same artist have the same Spotify genre tags.

Note

In addition to artists being the observational unit, Spotify provides no vote counts for tags on track basis. However, some track feature multiple artists. Thus, we calculated the vote for each track’s tags by combining all track artists’ tags and counting the number of tag appearences per track. 30,672 tracks (14%) featured multiple artists.
